# Tutorials: Guided Learning Paths 🚀

Welcome to the **AI4ALL-SRE Tutorials**. These guides are designed to take you from a fresh workstation to a high-fidelity, autonomous reliability laboratory.

---

## 🏗️ Learning Roadmap

Our tutorials follow a progressive complexity model, moving from infrastructure setup to advanced AI specialization.

### 1. [Quickstart: Deploying the Control Plane](01-quickstart.md)
**Level: Beginner**  
Deploy the core K3s cluster, Linkerd mesh, and Observability stack. This is the foundational "Golden Path" for all laboratory activities.

### 2. [Industrial MLOps Pipeline: AI Specialization](02-ai-model-finetuning.md)
**Level: Advanced**  
Transform a generic LLM into a specialized **SRE-Kernel**. Learn to manage local fine-tuning environments, quantization strategies, and verifiable reasoning benchmarks.

---

## 🎓 Success Criteria
Upon completion of these tutorials, an engineer should be capable of:
- Managing a local-first service mesh with mTLS.
- Triggering and observing multi-step chaos experiments.
- Iterating on autonomous remediation logic using specialized models.
